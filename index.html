Introduction to Git Tutorial

Welcome to this introductory tutorial on Git! Here, you'll learn the fundamental concepts and practical steps to get started with version control using Git, manage your code with GitHub, and track your coding time with Hackatime.

Introduction to GitHub

GitHub is a web-based platform that uses Git for version control. It's a central place where developers can store, manage, and collaborate on their code projects.

What is a repository?

A repository (often shortened to "repo") is like a project folder for your code. It contains all the files, folders, and revision history of your project. Think of it as a central place where all your project-related work is stored and organized. When you create a repository, you're essentially setting up a dedicated space for your project to live and be version-controlled.

Creating a repository

To create your first repository on GitHub:

  1. Go to GitHub.com: If you don't have an account, sign up for free.
  2. Click the "New" button: On the left sidebar of your GitHub dashboard, you'll see a green "New" button.
  3. Name your repository: Choose a descriptive name for your project (e.g., my-first-git-project, website-tutorial).
  4. Add a description (optional): Briefly explain what your project is about.
  5. Choose visibility: Select "Public" if you want anyone to see your code, or "Private" if you want to keep it to yourself or specific collaborators.
  6. Initialize with a README (recommended): Check the box to add a README file. This file is often used to describe your project.
  7. Click "Create repository": Your new repository will be created and ready for your code!

Introduction to VSCode

VS Code (Visual Studio Code) is a free, open-source code editor developed by Microsoft. It's incredibly popular among developers for its versatility, extensive features, and vast ecosystem of extensions.

How to install VSCode

Once VSCode is installed, you can open your newly created GitHub repository locally:

Clone the Repository

  1. On your GitHub repository page, click the green "Code" button.
  2. Copy the HTTPS URL.
  3. Open your terminal or command prompt.
  4. Navigate to the directory where you want to store your project:
    cd Documents/CodingProjects
  5. Type the following command and press Enter:
    git clone [PASTE_YOUR_GITHUB_URL_HERE]
    This will download a copy of your repository to your local machine.

Open in VSCode

  1. Once the cloning is complete, open VSCode.
  2. Go to File > Open Folder... (or File > Open... on some systems).
  3. Navigate to the folder that was just created by the git clone command (it will have the same name as your repository) and click "Select Folder".

VSCode will open your repository, and you'll see its files in the Explorer sidebar.
