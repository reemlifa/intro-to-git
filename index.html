Introduction to Git Tutorial

Welcome to this introductory tutorial on Git! Here, you'll learn the fundamental concepts and practical steps to get started with version control using Git, manage your code with GitHub, and track your coding time with Hackatime.

Introduction to GitHub

GitHub is a web-based platform that uses Git for version control. It's a central place where developers can store, manage, and collaborate on their code projects.

What is a repository?

A repository (often shortened to "repo") is like a project folder for your code. It contains all the files, folders, and revision history of your project.

Creating a repository

To create your first repository on GitHub:

  1. Go to GitHub.com: If you don't have an account, sign up for free.
  2. Click the "New" button: You'll see a green "New" button.
  3. Name your repository: Choose a descriptive name (e.g., my-first-git-project).
  4. Add a description (optional): Explain what your project is about.
  5. Choose visibility: Select "Public" or "Private".
  6. Initialize with a README: Recommended for describing your project.
  7. Click "Create repository": Your new repo is ready!

Introduction to VSCode

VS Code is a free, open-source code editor from Microsoft. It's popular for its versatility, features, and extensions.

How to install VSCode

Once VSCode is installed, open your GitHub repo locally:

Clone the Repository

  1. Click the green "Code" button on your GitHub repo.
  2. Copy the HTTPS URL.
  3. Open your terminal or command prompt.
  4. Navigate to your project folder: cd Documents/CodingProjects
  5. Run: git clone [PASTE_YOUR_GITHUB_URL_HERE]

Open in VSCode

  1. Open VSCode.
  2. Go to File > Open Folder...
  3. Select the folder that was just cloned.

You should now see your files in the Explorer sidebar.
